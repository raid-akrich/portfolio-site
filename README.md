Built from scratch using HTML, SCSS, and JavaScript (and later React if needed).
The goal is to showcase my projects, skills, and experience in a clean, modern, and professional way.

Features

	•	Clean, minimal design
	•	Sections for About, Projects, Contact
	•	Reusable components and organized file structure

Tech Stack
	•	HTML5
	•	SCSS / CSS3
	•	JavaScript
	•	(React will be added later if needed)

How to Run
	1.	Clone the repo
	2.	Open index.html in your browser
